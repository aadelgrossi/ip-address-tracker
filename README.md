## To run locally...

### Clone the repo
```bash
$ git clone https://github.com/aadelgrossi/ip-address-tracker
```

### Install dependencies
```bash
$ yarn install 
```

### Start the server
```bash
$ yarn dev
```

You should be able to access it on http://localhost:5000
